# subusharma-portfolio

A simple portfolio application built with Next.js 16, TypeScript, and Tailwind CSS for teaching basic React concepts.

- 

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open your browser** and navigate to [http://localhost:3000](http://localhost:3000)



## Building for Production

```bash
npm run build
npm start
```
